# Fonts of the estate

The book is set in the estate's canon faces, all released under the
SIL Open Font License 1.1 (redistribution permitted, which is why they
ride along here):

- **Cormorant Garamond** (display) — Christian Thalmann / Catharsis Fonts
- **Crimson Pro** (prose) — Jacques Le Bailly, after Sebastian Kosch
- **IBM Plex Mono** (the procedural voice) — Mike Abbink, Bold Monday, IBM
- **EB Garamond** (reserve) — Georg Duffner, Octavio Pardo

Install these before opening USING_THIS_MATTER.docx and the book will
render exactly as designed. Without them, your word processor will
substitute; the words survive, the voice thins.
